# titech-sokoban
人工知能の授業用に作った倉庫番ソルバ

## 使い方
### マップの作成
テキストファイルで作る。

- `#`: 障害物。いくつでも。
- `$`: 荷物。いくつでも。
- `.`: 格納場所。荷物と同じ数だけ。
- `@`: ロボット。1つだけ。

必ず閉空間にロボットを配置すること。

(例)
```
 #####
 #.. #
###  #
# $  #
# $ ##
#@  #
#####
```

### 使い方
こんな感じ。
```
$ ./astar_search.py sample/01.txt
```
内容は読めば分かる。
